HOW TO OPEN — IT'S VERY SIMPLE
-------------------------------
1. Open the "JoyVet-CDS" folder
2. Double-click "index.html"
   → It opens automatically in your web browser
3. That's it — no installation, no internet needed!

COMPATIBLE BROWSERS (use any of these)
---------------------------------------
• Google Chrome   (recommended)
• Microsoft Edge
• Mozilla Firefox
• Apple Safari

FILES IN THIS FOLDER
--------------------
  index.html        — Main application  ← OPEN THIS FILE
  knowledge-base.js — Veterinary diagnosis engine (must stay in same folder)
  README.txt        — This file

  ⚠ Keep index.html and knowledge-base.js in the SAME folder always.

HOW TO SHARE TO OTHER DEVICES
------------------------------
• USB drive:    Copy the "JoyVet-CDS" folder to a USB stick.
                Plug into any device → open index.html.

• LAN / Network: Put the folder on a shared network drive.
                 Open index.html from any computer on the network.

• Email / Cloud: Zip the whole folder → send → unzip → open index.html.

• Tablet (iPad): Transfer the folder via AirDrop or Files app.
                 Tap index.html to open in Safari.

DATA & PRIVACY
--------------
• All case data is stored ONLY on your device (browser localStorage)
• Nothing is ever sent over the internet
• Case history saves the last 10 sessions automatically
• Each device stores its own history separately

KEYBOARD SHORTCUTS
------------------
• Ctrl+Enter    — Advance to next step
• Escape        — Go back one step
• ⚡ Quick Normal button — Auto-fills section as "within normal limits"
